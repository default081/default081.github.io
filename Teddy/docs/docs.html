Teddy Template Description

1- Template Structure

  • The main folder contains 4 html files

    - index.html : File for viewing demo templates

    - paralax.html : Version of the template with parallax effect

    - particles.html : Version of the template with particles effect

    - standard.html : Version of the template with static header

  • The css folder contains styles and sass instructions for the template

    - The libs folder contains styles for plugins and libraries

  • The JS folder contains scripts and configuration files for particles plugin

    - The libs folder contains js scripts for plugins and libraries

  • The img folder contains pictures and icons

2- How to edit template

  • How to edit header image: Upload your photo to the img folder, then open the style.css file (located in the css folder), find the .header instruction, and replace the value in the url function
    Example: url(../img/photo-title.jpg)

  • Change other image: Upload your photo to the img folder, open the html file and find the img tag, then go to the style.css file and replace the name of the photography

  • How to change fonts: Find the font you want on Google Fonts, press the button "Select this font", copy the link tag, and insert into the header section, then use the command font-family: ;
    Example: <link href="https://fonts.googleapis.com/css?family=Roboto" rel="stylesheet">
